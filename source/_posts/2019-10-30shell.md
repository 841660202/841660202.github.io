## 删除某个目录下所有子目录的node_modules

```
# 查找 /tmp/test/ 子目录下所有 modu 执行删除
find /tmp/test/ -type d -name modu |xargs rm -rf
```

```
# 查找 /tmp/test/ 子目录下所有 node_modules 执行删除
find /tmp/test/ -type d -name node_modules |xargs rm -rf
```
